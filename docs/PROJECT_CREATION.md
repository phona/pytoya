# Project Creation Guide

PyToYa uses a guided wizard to create projects and set up extraction defaults.

## Wizard Steps
1. **Basics**: Enter project name and optional description.
2. **Strategy**: Choose schema-based or prompt-based extraction.
3. **Schema or Prompt**:
   - Schema-based: select an existing schema or build a new one inline.
   - Prompt-based: describe extraction needs and generate an optimized prompt.
4. **Models**: Pick default OCR and LLM models (optional).
5. **Review**: Confirm selections and create the project.

## Tips
- Schema-based projects can still add prompts later; prompt-based projects can still attach schemas later.
- Use validation scripts in the project detail page to enforce custom rules after extraction.
